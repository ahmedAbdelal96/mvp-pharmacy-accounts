# MVP — 01 Product Scope

## الهدف

بناء نظام صغير وسريع لإدارة حسابات الأطراف داخل الصيدلية.

النسخة الأولى يجب أن تجاوب على:

```txt
مين عليه فلوس؟
مين له فلوس؟
دفعنا لمين؟
قبضنا من مين؟
كشف حساب كل طرف إيه؟
```

## المستخدمين

```txt
صاحب الصيدلية
محاسب الصيدلية
مدير الصيدلية
```

## MVP Features

### 1. Dashboard

يعرض:

```txt
إجمالي العملاء عليهم
إجمالي الموردين لهم
عدد العملاء
عدد الموردين
عدد شركات الأدوية
آخر الحركات
```

### 2. Parties

إدارة الأطراف:

```txt
الاسم
النوع
الهاتف
العنوان
مسؤول التواصل
ملاحظات
نشط / غير نشط
```

### 3. Account Transactions

كل حركة مرتبطة بطرف:

```txt
الطرف
تاريخ الحركة
نوع الحركة
الاتجاه: له / عليه
المبلغ
الوصف
رقم مرجعي اختياري
ملاحظات
```

### 4. Party Statement

كشف الحساب يعرض:

```txt
التاريخ
نوع الحركة
الوصف
له
عليه
الرصيد بعد الحركة
ملاحظات
```

### 5. Reports

تقارير بسيطة:

```txt
تقرير الأطراف التي عليها
تقرير الأطراف التي لها
تقرير أرصدة كل الأطراف
تقرير الحركات
```

## Out of Scope

لا تنفذ في V1:

```txt
Products
Inventory
Branches
Warehouses
Sales invoice items
Purchase invoice items
POS
Barcode
Tax
COGS
Accounting journal
Cashbox
Reconciliation
```

## لغة الواجهة

استخدم:

```txt
له
عليه
دفع
تحصيل
تسوية
كشف حساب
الرصيد الحالي
```

وتجنب في الواجهة:

```txt
Debit
Credit
Journal
Ledger
Trial Balance
```

## قاعدة النطاق

```txt
أي ميزة لا تخدم الأطراف والحركات وكشف الحساب ليست ضمن V1.
```
